This is a simple project implementing a very simple verison of wave function collapse. It have a total of 12 tiles the algorithm can choose to fill a grid, that can be of any size within a 1 to 100 rows and columns. 
It have some ui to the user to togle, like a display grid index, display entropy, or slow the algorithm down. It meanly use javascript, and p5 library to do some web utilites.
